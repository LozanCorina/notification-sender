## About Laravel

We should notify user via FCM and Vonage and disable future notifications via specific channel if it was failed

In case delivery failed:
FCM: set user->push_notifications_token = null
Vonage: if phone number not in supported network then we set user->phone_unreachable = true otherwise we write error to log

Send to a particular user or bulk

Using: 
"laravel-notification-channels/fcm"
"laravel/vonage-notification-channel"
